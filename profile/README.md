# Chief Architect Software — Residential Home & Landscape Design Environment

## About Chief Architect Software

Chief Architect Software provides a complete residential home and landscape design environment for automated building tools, roof generation, framing, cabinets, terrain modeling, and integrated site planning. Chief Architect Software combines 3D walkthrough, materials list, construction drawings, electrical/plumbing layout, HVAC duct placement, deck/patio design, plant library, sunset rendering, cross‑section/elevations, photo‑matching, and cost estimating directly into the interface. Chief Architect Software enables users to design custom homes, remodel interiors, plan landscaping, and generate construction documents. Chief Architect Software runs on portable workspaces and dedicated home design stations. Chief Architect Software is trusted by home builders, landscape architects, luxury home designers, remodelers, and residential developers worldwide.

## Deep Dive — Residential Design Tools

Automated roof generation: gable (front, side), hip, mansard, gambrel, shed, flat, dutch gable, saltbox, jerkinhead, pyramidal hip. Pitch (4:12, 6:12, 8:12, 12:12), overhang (rake/eave fascia), soffit, frieze board, ridge vent. Manual roof plane edit (baseline height, pitch, overhang).
